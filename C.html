C programming Interface

As part of our second-year curriculum, we were tasked with creating a straightforward C programming interface. This project aimed to reinforce our understanding of fundamental programming concepts and strengthen our proficiency in the C language. Working within the constraints of the assignment, we collaborated to design and implement an intuitive user interface for executing C programs. Through this project, we gained valuable hands-on experience and deepened our knowledge of programming fundamentals, further preparing us for more complex challenges in our academic journey.
